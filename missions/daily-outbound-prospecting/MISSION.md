# Daily Outbound Prospecting

Send personalized cold outreach to qualified prospects every morning.

## Steps

1. Pull today's prospect batch from CRM
2. Research each prospect's company and recent activity
3. Draft personalized cold emails with objection preemption
4. Schedule multi-touch follow-up sequences
5. Log all outreach activity to CRM

## Prompt

Use #Prospect research to review today's prospect batch from the CRM. Use #Email personalization to craft individualized cold emails referencing each prospect's company, recent news, and pain points. Apply #Objection handling frameworks to preempt common pushbacks in the email body. Use #Follow-up sequences to schedule automated follow-ups at day 3, day 7, and day 14. Log all activity to the CRM using #CRM management.
